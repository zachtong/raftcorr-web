# RAFTcorr — Project Website

Promotional website for **RAFTcorr**: a deep learning framework for Digital Image Correlation (DIC).

**Live:** [zachtong.github.io/raftcorr](https://zachtong.github.io/raftcorr)

## Development

No build tools required. Open `index.html` in a browser or use a local server:

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```

## Structure

```
├── index.html          # Single-page website
├── style.css           # All styles (CSS variables, responsive)
├── script.js           # Scroll animations, carousel, interactions
└── assets/
    ├── videos/         # Hero animation, demo video
    ├── images/         # Results, authors, method diagrams
    └── icons/          # SVG icons (paper, code, data)
```

## Deployment

Deployed via GitHub Pages from the `main` branch root.
